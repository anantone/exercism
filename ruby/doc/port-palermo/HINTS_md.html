    Hints¶ ↑

    1. Create the ports identifier¶ ↑

    • You need to define a constant that should contain the symbol value specified in the instructions.

    2. Get identifiers for other ports¶ ↑

    • A string can be slice to get parts of it.

    • Strings have a method to upcase the letters.

    • Strings have a method to convert a string to a symbol.

    3. Determine which terminal a ship should dock at¶ ↑

    • Symbol has a method to convert a symbol to a string.

    • A string can be slice to get parts of it.

    • Use conditional {if}, to check if the ship is carrying "OIL" or "GAS".
